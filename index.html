Initialize the map.
Call the initializeMap() method to display the map.

Note that this function takes a parameters and it's about:

Code 
initializeMap(arrayOfDisplayedGraphics, withSketch = false, HTMLElementsArray=[], AddModalId="", HasLabel=false, HasKML=false)

// arrayOfDisplayedGraphics:  The array of the locations to display on the map (Array of objects),
// withSketch : Whether to have sketch widget or not(Boolean),
// HTMLElementsArray: All the html components to add in the map(Array of HTML elements[HTMLElement,Boolean]),
// HTMLElementsArray[(HTMLElement), button will replace the search elemnt or not (Boolean)],
// HTMLElementsArray will send it to the addWidgets function that called inside the initializeMap function,
// AddModalId: Uique ID for the html modal to perform the action of add  new location (String),
// HasLabel: Whether to display label for each location or not(Boolean),
// HasKML: Whether the project has kml locations or not  (Boolean)
